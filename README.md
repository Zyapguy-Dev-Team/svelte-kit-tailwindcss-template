NOTE: This repo uses PNPM instead of NPM.

## Install PNPM
```bash
npm i -g pnpm@latest
```

## Install latest dependencies
```bash
pnpm update
```

## Run in dev mode (Hot Module Reload and Tailwind JIT mode enabled)
```bash
pnpm dev
```

## Build
```bash
pnpm build
```

## Preview production build
```bash
pnpm preview
```

## Start production server
```bash
pnpm start
```
